Bitcoin, Litecoin and Dollar exchange plugin for WordPress.

== Description ==

xCoinMoney exchange plugin allows you turn your WordPress site in Bitcoin/Litecoin/Dollar exchanger effortlessly with fully adjustable commissions and detailing for each transaction.
All you need to do is to install and activate the xCoinMoney Exchange plugin for your online store, <a href="https://www.xcoinmoney.com/auth/signup" rel="nofollow">SignUp at xCoinMoney.com</a> as a Business account and specify your User ID and API key in the plugin settings.<br />
To create a new page in WordPress with Exchange form, there is a need to use short codes:
<ul><li>[xcoinmoney_exchange] - form of Ercahnger.</li>
<li>[exchange_rates] - current exchange rates.</li>
<li>[available_balance] - Available balances of Exchanger. This is also the balance in the Merchant’s xCoinMoney account.</li></ul>
Full instruction how to configure and start Exchanger see at <a href="https://www.xcoinmoney.com/plugins/wordpress-exchange">xCoinMoney plugin page</a>.

== Installation ==

1. <a href="https://www.xcoinmoney.com/auth/signup" rel="nofollow">SignUp</a> at xCoinMoney.com as a Business account for free
1. Enable API in <a href="https://www.xcoinmoney.com/user/settings" rel="nofollow">User settings</a>
1. Upload `xcoinmoney-free-cryptocurrency-exchange` folder to the `/wp-content/plugins/` directory of your online store and activate the plugin.
1. Go to `xCoinMoney Exchange` → `xCoinMoney Exchange`.
1. In `Users must be registered` select which users can use the exchanger (all or only registered).
1. Copy 'Your callback URL' from the plugin settings in WordPress into your xCoinMoney user Settings 'Callback URL'
1. Copy "API key" and "Merchant ID" from xCoinMoney user Settings to the WordPress Exchange plugin settings
1. Save all changes
